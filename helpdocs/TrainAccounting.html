This page accepts the per-train ticket numbers and receipts. The end-of day information which is not train specific is entered on the Daily Accounting page.

Note that this train window is a popup. When finished, press Save Changes and then, if the changes were accepted, press Return to Daily to remove the train window.

Ticket Numbers Ticket beginning and ending serial numbers are entered here. If there is a gap in the ticket sequence, enter them any way and document the gap in the Daily Accounting comment field.

Receipts The cash register categories should be copied verbatim.
