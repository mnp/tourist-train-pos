Red labelled fields must be filled in. State OR Country must be filled--Country of US is assumed if State is filled.

You can search for an approximate value of a field with "wildcard" searching. A "%" character means match anything. Eg, to find all values starting with S, type "s%" into the field.

You can use the TAB key to move to the next field in the form. Shift Tab move to previous field.

When in a pulldown field, (Eg, State, Country...) you can press the first letter of the item and it will jump there.
